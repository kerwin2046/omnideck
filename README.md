# 🚀 OmniDeck.nvim

> Your Personal Cyberdeck for Modern Development.

OmniDeck.nvim is my personal development environment, designed for speed, productivity, and a beautiful terminal workflow.

It combines the power of LazyVim with a carefully curated collection of tools and configurations to create a consistent experience across macOS and Linux.

## ✨ Features

* ⚡ Powered by LazyVim
* 💤 Lazy-loaded plugins for fast startup
* 🎨 Modern UI and developer-friendly defaults
* 🔍 Telescope, FZF, Ripgrep integration
* 🌳 Treesitter syntax highlighting
* 🤖 AI-assisted development workflow
* 🐙 Git & Lazygit integration
* 📦 Mason-powered LSP management
* 🚀 Optimized terminal experience

## 🛠 Included Ecosystem

OmniDeck extends beyond Neovim and aims to provide a complete workstation setup.

* Neovim
* Kitty
* Zsh
* Starship
* Fastfetch
* Git
* Lazygit
* FZF
* Ripgrep
* FD

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/yourname/omnideck.nvim.git ~/.config/nvim
```

Start Neovim:

```bash
nvim
```

LazyVim will automatically install all required plugins.
